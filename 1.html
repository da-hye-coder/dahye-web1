HTML

Hypertext Markup Language, a standardized system for tagging text files to achieve font, color, graphic, and hyperlink effects on World Wide Web pages.

Short for HyperText Markup Language, the authoring language used to create documents on the World Wide Web. HTML is similar to SGML,although it is not a strict subset. HTML defines the structure andlayout of a Web document by using a variety of tags and attributes.
